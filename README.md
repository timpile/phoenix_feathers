# Phoenix Feathers

**CURRENTLY IN DEVELOPMENT**

A UI component library built for Phoenix LiveView.

## Installation

The package can be installed by adding `phoenix_feathers` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:phoenix_feathers, "~> 1.0.2-alpha"}
  ]
end
```
